# Atividade: Desenvolvimento da Página Index

## Descrição
Este projeto tem como objetivo criar a página inicial (index) do site [LH Games]. A página será responsiva e terá como foco principal [Vendas de Jogos e Consoles].

## Tecnologias Utilizadas
* **HTML5 e CSS3:** para marcação e estilização;
* **Media query e bootstrap:** para Implementação de responsividade;
* **JQuery e JavaScript:** para implementação de funcionalidades.
